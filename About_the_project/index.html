Welcome! My name is Idan Hajbeko,
and I'm the creator of this website
Hashing, encryption, decryption, and everything in the field of cryptography have always interested me.
Another thing that is interesting to me is bitwise operations digital logic.

Not so long ago, I read about how AES works, and it blew my mind.
Something you need to know about me: after I read something that blows my mind, I have to do something about it until I'm satisfied.
To satisfy my curiosity after I read it, I wanted to create my own hashing algorithm and encryption algorithm, so I did.
But it wasn't enough. I needed to do more, so I decided to combine my love of bitwise operations and cryptography.

So I've created this website.
Each level presents you with an input box where you need to try different text inputs to find the custon encryption or hashing algorithm used,
and ultimately discover the original string from the given cipher.

I hope this website will introduce you to the fascinating world of bitwise operations, digital logic, and cryptography.
I hope you have fun exploring and learning new things!.
